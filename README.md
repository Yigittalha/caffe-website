# CAFFE - Premium Coffee Shop Website

A modern, mobile-responsive single-page coffee shop website built with React and Tailwind CSS.

![Coffee Website](https://images.unsplash.com/photo-1447933601403-0c6688de566e?w=800&h=400&fit=crop&auto=format)

## 🌟 Features

- 📱 **Mobile Responsive**: Perfect display across all devices
- ☕ **Coffee Menu**: Detailed coffee varieties with prices and images
- 🍰 **Dessert Menu**: Homemade desserts and descriptions
- 📞 **Contact Form**: Working contact form for customer messages
- 🎨 **Modern Design**: Coffee-themed color palette and typography
- 🔗 **Social Media**: Social media integration
- ⚡ **Fast Navigation**: Smooth scroll animations
- 🎁 **Special Offers**: Dynamic promotional sections
- 📸 **Gallery**: Showcase of cafe atmosphere
- 👥 **Team Section**: Meet our professional team
- 💬 **Customer Testimonials**: Rotating customer reviews
- 📧 **Newsletter**: Email subscription system
- 🎭 **Animations**: Hover effects and smooth transitions
- 📊 **Live Counters**: Animated statistics

## 🛠 Technologies

- **React 18** - Modern JavaScript library
- **Tailwind CSS 3** - Utility-first CSS framework
- **Font Awesome** - Icon library
- **Google Fonts** - Inter font family
- **Unsplash API** - High-quality images

## 🚀 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yigittalha/caffe-website.git
   cd caffe-website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm start
   ```

4. **Open in browser:**
   [http://localhost:3000](http://localhost:3000)

## 🎨 Customization

### Color Palette
Custom coffee-themed colors defined in Tailwind config:
- `coffee-brown`: #8B4513
- `coffee-dark`: #3C2415
- `coffee-light`: #D2B48C
- `cream`: #F5F5DC

### Menu Content
Update menu items in `src/App.js` by modifying the `coffeeMenu` and `dessertMenu` arrays.

### Contact Information
Update contact details in the Contact Section of `src/App.js`.

### Images
All images are sourced from Unsplash with optimized URLs. Replace image URLs in the component arrays to customize visuals.

## 🌐 Deployment

### GitHub Pages
1. Go to repository Settings
2. Navigate to Pages section
3. Select "GitHub Actions" as source
4. Automatic deployment will be enabled

### Vercel
1. Connect your GitHub repository
2. Automatic build and deployment
3. Custom domain support

### Netlify
1. Connect repository
2. Build command: `npm run build`
3. Publish directory: `build`
