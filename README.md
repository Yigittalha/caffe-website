## ğŸ¨ Customization

### Color Palette
Custom coffee-themed colors defined in Tailwind config:
- `coffee-brown`: #8B4513
- `coffee-dark`: #3C2415
- `coffee-light`: #D2B48C
- `cream`: #F5F5DC

### Menu Content
Update menu items in `src/App.js` by modifying the `coffeeMenu` and `dessertMenu` arrays.

### Contact Information
Update contact details in the Contact Section of `src/App.js`.

### Images
All images are sourced from Unsplash with optimized URLs. Replace image URLs in the component arrays to customize visuals.
